# TOEIC 难点总结：词汇 · 短语 · 搭配 · 语法

> 本文整理 TOEIC 考试中出现频率高、但容易出错的语言点。适用于 Part 5/6（语法词汇）和 Part 7（阅读），同时也覆盖 Part 3/4 听力中常见的商务表达。

## 一、易混淆词汇（Part 5 高频陷阱）

### 1.1 形近词 / 音近词

| 词组 | 含义区分 | 例句 |
|---|---|---|
| **assure / ensure / insure** | 使（人）放心 / 确保（事） / 投保 | We **assure** you that... / **ensure** quality / **insure** a building |
| **adopt / adapt / adopt** | 采用 / 适应 / 收养 | **adopt** a policy / **adapt** to a new system |
| **complement / compliment** | 补充、衬托 / 称赞 | wine **complements** the meal / pay sb a **compliment** |
| **principal / principle** | 主要的；校长；本金 / 原则 | the **principal** reason / against my **principles** |
| **stationary / stationery** | 静止的 / 文具 | remain **stationary** / order **stationery** |
| **economic / economical** | 经济（学）的 / 节约的 | **economic** growth / an **economical** car |
| **considerable / considerate** | 相当大的 / 体贴的 | a **considerable** amount / **considerate** of you |
| **respectful / respective / respectable** | 恭敬的 / 各自的 / 体面的 | **respective** departments |
| **successful / successive** | 成功的 / 连续的 | three **successive** years |
| **industrial / industrious** | 工业的 / 勤奋的 | **industrial** zone / an **industrious** employee |
| **confident / confidential** | 自信的 / 机密的 | **confidential** information |
| **comprehensive / comprehensible** | 全面的 / 可理解的 | a **comprehensive** report |
| **sensitive / sensible** | 敏感的 / 明智的 | a **sensible** decision |
| **effective / efficient** | 有效的（产生效果） / 高效的（省时省力） | an **efficient** worker |
| **personal / personnel** | 个人的 / 全体员工（不可数）| the **personnel** department |
| **alternate / alternative** | 交替的；候补的 / 可替代的；备选方案 | an **alternative** route |
| **continual / continuous** | 反复发生的 / 不间断的 | **continuous** improvement |

### 1.2 词性陷阱（Part 5 最常考）

TOEIC Part 5 一半以上的题考"在空格里填什么词性"。判断方法：

- 冠词/形容词后 → **名词**：`a careful **selection**`
- 名词前 → **形容词**：`a **careful** review`
- 动词/形容词后 → **副词**：`work **carefully**`
- 助动词/be 之后 → **动词**或**分词**：`is **scheduled** to begin`
- 冠词 + ___ + 名词 → **形容词** 或 **现/过去分词**

常考"同根三件套"：
- *succeed → success → successful → successfully*
- *compete → competition / competitor / competitiveness → competitive → competitively*
- *produce → product / production / productivity → productive → productively*
- *expand → expansion → expansive / expandable*
- *apply → application / applicant / applicability → applicable*

| **subscribe to** | 订阅；同意 |
| **specialize in** | 专门从事 |

| **prevent / prohibit / discourage … from doing** | 阻止做某事 |

## 四、商务高频短语（Part 7 阅读必备）

- **come up with** 想出 / **come across** 偶遇 / **come into effect** 生效
- **carry out** 执行 / **carry on** 继续 / **carry over** 延续到（金额、休假）
- **set up** 建立 / **set out** 出发；阐述 / **set aside** 留出（时间、钱）/ **set back** 推迟；耽搁
- **take over** 接管 / **take on** 承担（任务）；雇用 / **take off** 起飞；脱掉；火爆起来 / **take into account** 考虑到
- **put off** 推迟 / **put up with** 忍受 / **put forward** 提出 / **put together** 拼凑、组织
- **turn down** 拒绝；调低 / **turn out** 结果是 / **turn in** 提交 / **turn over** 营业额达到
- **bring up** 提及 / **bring about** 引起 / **bring forward** 提前
- **call off** 取消 / **call on** 拜访；号召 / **call for** 需要、要求
- **look into** 调查 / **look over** 检查 / **look up to** 尊敬
- **run out of** 用完 / **run into** 偶遇；遇到（问题）/ **run for** 竞选
- **be on track to** 有望（按时完成）
- **in light of** 鉴于 / **on behalf of** 代表 / **in lieu of** 代替
- **with regard to / regarding / concerning** 关于
- **as opposed to** 而不是 / **as of (date)** 自（某日）起
- **at the latest** 最迟 / **at one's earliest convenience** 在您方便时尽快
- **on short notice** 临时通知 / **on a regular basis** 定期地
- **be poised to do** 准备好做 / **be slated to do** 预定做

## 五、语法重点

### 5.1 时态

- **现在完成时**：`since / for / so far / over the past X years / recently / lately / already / yet / just`
  - *Sales **have increased** by 10% **over the past quarter**.*
- **过去完成时**：表"过去的过去"，常配 `by the time / before / when` + 过去时从句
  - *By the time we arrived, the meeting **had already started**.*
- **将来完成时**：`by + 将来时间点`
  - *By next month, we **will have completed** the project.*
- **将来进行时**（礼貌口吻，预定计划）：
  - *Mr. Lee **will be attending** the conference next week.*

### 5.2 主谓一致（高频考点）

- `one of + 复数名词 + 单数动词`：*One of the proposals **is** under review.*
- `the number of` 单数 / `a number of` 复数
- `each / every / either / neither + 单数`
- `(both) A and B` 复数 / `A as well as B` 跟随 **A** / `either A or B` 跟随 **B**
- 集合名词（committee, staff, audience）在 TOEIC 中通常 **视为单数**

### 5.3 分词作定语（Part 5、6 经典）

- **现在分词**（主动 / 进行）：*the **rising** demand* / *the issue **affecting** sales*
- **过去分词**（被动 / 完成）：*the **revised** schedule* / *items **purchased** last week*

判断口诀：**主动用 -ing，被动用 -ed**。

### 5.4 关系代词与关系副词

| 词 | 先行词 | 在从句中作 |
|---|---|---|
| who / whom | 人 | 主语 / 宾语 |
| whose | 人或物 | 定语 |
| which | 物 | 主语 / 宾语 |
| that | 人或物 | 主/宾（不用于非限定）|
| where | 地点 | 状语 |
| when | 时间 | 状语 |
| why | reason | 状语 |

陷阱：`a place **where** he works` ✅ / `a place **which** he works **in**` ✅，但不能 `a place which he works`（缺介词）。

### 5.5 虚拟语气

- **建议、要求、命令** 类动词后 that 从句用 `(should) + 动词原形`：
  - *We require that he **(should) submit** the report by Friday.*
  - 类似动词：suggest, recommend, propose, demand, request, insist, require, order, ask
- **if 虚拟**：
  - 与现在事实相反：*If I **were** you, I **would** take the offer.*
  - 与过去事实相反：*If we **had ordered** earlier, we **would have received** it on time.*

### 5.6 倒装

- **否定副词置首**：*Not only **did sales increase**, but profits also rose.*
- **only + 状语置首**：*Only after the renovation **did the store reopen**.*
- **so / neither 倒装**：*So **did** the manager.*

### 5.7 比较级与最高级

- `as + 形容词原级 + as` / `not as / so + 原级 + as`
- 比较级强调：**much / far / considerably / significantly / even / a lot** + 比较级
  - ⚠️ **very** 不能修饰比较级
- `the + 最高级 + (名词) + 范围`：*the most reliable supplier **in the region***
- `no other + 单数名词 ... as ... as / 比较级 than`：*No other product is **as durable as** ours.*

### 5.8 连词 vs 介词 vs 副词（最易错）

| 含义 | 连词（+ 句子）| 介词（+ 名词）| 副词（+ 句子但要分号 / 句号）|
|---|---|---|---|
| 因为 | because, since, as | because of, due to, owing to | — |
| 尽管 | although, though, even though | despite, in spite of | nevertheless, nonetheless, however |
| 如果 | if, provided that, as long as | in case of | — |
| 在……期间 | while | during | meanwhile |
| 所以 | so | — | therefore, thus, hence, consequently |

> ⚠️ TOEIC 经典考法：题干提供 `_____ the bad weather, the event was held.` —— 选 **Despite**（介词），不能用 Although（连词）。

### 5.9 不定式 vs 动名词

- **只接不定式**：want, hope, plan, decide, agree, promise, refuse, manage, fail, offer, intend, aim
- **只接动名词**：enjoy, finish, avoid, mind, suggest, recommend, consider, postpone, deny, admit, risk, practice, give up, look forward to
- **意义不同**：
  - *stop to do* 停下来去做 / *stop doing* 停止做
  - *remember to do* 记得去做 / *remember doing* 记得做过
  - *forget to do* 忘记要做 / *forget doing* 忘记做过

### 5.10 介词 by / until / within / by the time

- **by + 时间点**：在……之前（一次性动作）—— *Submit it **by** Friday.*
- **until + 时间点**：直到……（持续动作）—— *The office is open **until** 9 p.m.*
- **within + 时间段**：在……之内 —— *Reply **within** 3 business days.*
- **by the time + 从句**：到……时（主句多用完成时）

## 七、常考"假朋友"——以为认识却选错

- **regard** ≠ 注视；常用 *with regard to*（关于）
- **address** 在邮件中 = "解决、提及"，不是"地址"
- **issue** 多义：问题；期刊一期；签发（issue a refund）
- **subject** 名词=主题；动词=使遭受；形容词 *be subject to*=取决于
- **practice** 名词：惯例、做法（business practice）
- **claim** 动词：声称；索赔（claim insurance）
- **observe** 不只是"观察"，也指"遵守"（observe the regulation）
- **honor** = 兑现（honor a warranty / the coupon）
- **entertain** = 考虑（entertain a proposal）
- **second** 动词 = 附议（second the motion）
- **table** 动词 = 搁置（英式）/ 提交讨论（美式）
- **dated** 形容词 = 过时的；标有日期的
- **outstanding** = ① 杰出的 ② 未付的（outstanding balance）
- **board** = 董事会；登机/上车（board the flight）
- **draft** = 草稿；汇票
- **post** = 职位；岗位；张贴

## 八、备考小贴士

1. **词性优先**：Part 5 看到选项是同根四件套，先用语法判断词性，再判含义。
2. **介词背搭配**：动词 + 介词、名词 + 介词、形容词 + 介词三类要分别整理。
3. **同义替换**：Part 7 答案多为题干同义改写，例如 *purchase → buy*、*assist → help*、*proceed → continue*、*relocate → move*。
4. **听力关键词**：notice、announcement、reminder 类文本开头常有 *Attention please / This is a reminder that / We regret to inform you that*。
5. **时间数字**：Part 3/4 常考 "what time / when / how much / how many"，听到即记。
6. **不要被生词卡住**：TOEIC 不考偏词，多数难词都在商务八大场景里反复出现，多刷真题最有效。
