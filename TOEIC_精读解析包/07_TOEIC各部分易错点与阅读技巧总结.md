# 第一部分：听力易错点（Part 1–4）

## Part 1 照片描述（6 题）

### 易错点
1. **被动 vs 进行时混淆**
   - `is being + 过去分词` 表示"正在被…"，**必须画面里有人在做这个动作**。
   - 没人就只能用 `has been + 过去分词`（结果状态）。
   - ❌ 看到桌上摆好的杯子选 *Cups are being arranged*（没人摆）
   - ✅ *Cups have been arranged on the table*

2. **高级替换词**
   - *examining* = 检查 / *inspecting* = 查验 / *adjusting* = 调整 / *operating* = 操作
   - *be displayed / be stacked / be mounted / be hung / be propped against*

3. **看着像但没拍到的陷阱**
   - 选项提到一个画面里没有的物品（楼梯、车、人）—— 直接排除。

### 处理动作
- 听之前**先扫照片**：场景（办公室/工厂/户外/餐厅）、人数、动作。
- 听到选项**先排除**：人物动作不符 → 排；物体不存在 → 排；时态不符 → 排。

## Part 2 应答问题（25 题）—— TOEIC 听力**最难**部分

### 五大陷阱
1. **同音/近音陷阱**：问题里有 `train` → 选项里出现 `training` 引诱你选错。
2. **重复词陷阱**：选项重复问题的关键词，几乎都是错的。
3. **时态错位**：问"昨天去哪了"，错答案用现在时回答。
4. **WH 问题答 Yes/No**：100% 是错的（除非是 `Do you know who…`）。
5. **似是而非的话题转移**：听起来相关其实在答别的问题。

### 间接应答（高频考点）
TOEIC 越来越喜欢"不直接回答"的答案：

| 问题 | 间接应答 |
|---|---|
| When does the meeting start? | **I'm not sure, let me check.** |
| Where's the manager? | **Isn't she in a meeting?** |
| Did you finish the report? | **It's on your desk.** |
| Who's going to the conference? | **The list hasn't been finalized.** |
| How do I get to the lobby? | **I'll show you.** |

### 处理动作
- **听首词**：WH 词决定答案方向（Who→人 / Where→地点 / When→时间 / Why→原因 / How→方式或程度）。
- **遇到陈述句、选择疑问句、反问句**，要听完整。
- **"I don't know / Let me check / It depends"** 是万能正确答案的信号。

## Part 3 对话（39 题，13 段 × 3 题）

### 易错点
1. **三道题的顺序 = 对话顺序**（开头、中间、结尾），不会乱跳。
2. **第一题常考**：
   - 说话人身份 / 关系 / 地点 / 对话目的
   - 答案在**开头两句**
3. **第二题常考**：细节（数字、日期、问题原因）
4. **第三题常考**：
   - "What will the speaker do next?" → 答案在**结尾**
   - "What does the man imply?" → **意图题**

### 图表题（Look at the graphic）
- **绝对禁忌**：图表上明示的信息（如价格、日期、楼层），**不会**在听力里直接念出来。
- **听变更**：折扣后、被排除的、被推荐的、剩下的那个。
- 提前读图表 **2 秒**：四个选项分别对应图表里的什么。

### 意图题（What does the speaker mean by ...?）
- 不要翻译字面意思！
- 听上下文："为什么这么说" → 抱怨 / 同意 / 拒绝 / 提议 / 暗示忙不过来。

### 三人对话
- 听清 **谁说了什么**：通常一男两女或一女两男。
- 题目会问 "What does the woman with the dark suit say...？" 类指代要追。

## Part 4 短独白（30 题）

### 高频文本类型
1. **电话留言（voicemail）**：Hi, this is X from Y. I'm calling about…
2. **公司广播 / 通知（announcement）**：Attention, employees / passengers / shoppers…
3. **会议节选（meeting excerpt）**：Thanks for coming. Today we'll discuss…
4. **广告（advertisement）**：Looking for…? Come to…
5. **导览 / 介绍（tour / introduction）**：Welcome to… / It's my pleasure to introduce…
6. **新闻 / 天气 / 交通报道**

### 易错点
- **听首句定身份**：tour guide / radio host / store manager / receptionist
- **数字陷阱**：百分比、价格、日期、电话号码 —— 听到立即标记
- **同义改写**：原文 *purchase* → 选项 *buy*；原文 *complimentary* → 选项 *free*；原文 *malfunction* → 选项 *not working*

# 第二部分：阅读易错点（Part 5–7）

## Part 5 单句语法/词汇（30 题）—— 节奏题

**建议用时：每题 15–20 秒**，30 题 ≤ 10 分钟。

### 解题流程
```
看选项 → 判断题型（词性 / 词汇 / 语法）
   ↓
词性题：看空格前后 → 判断需要 名/动/形/副
   ↓
词汇题：把四个词代入 → 看搭配 / 含义
   ↓
语法题：看时态信号 / 主谓一致 / 连词介词
```

### 五大题型 & 易错点

#### ① 词性题（约 1/3）
**信号**：选项是同根词四件套（succeed / success / successful / successfully）

| 空格位置 | 词性 |
|---|---|
| 冠词/代词 + ___ + 名词 | 形容词 |
| 动词/形容词 + ___ | 副词 |
| 冠词/形容词 + ___ | 名词 |
| be / 助动词 + ___ | 动词/分词 |
| 介词 + ___ | 名词/动名词 |

**易错**：副词修饰整个句子时位置灵活 —— *Unfortunately, the shipment was delayed.*

#### ② 词汇题
**易错**：四个选项词性相同、含义相近。例如 *accept / receive / acknowledge / admit*。
**对策**：看**搭配** —— *accept an offer / receive a package / acknowledge receipt / admit a mistake*。

#### ③ 时态题
**信号词决定时态**：
- `since / for / so far / over the past / recently` → 现在完成
- `by next month / by 2030` → 将来完成
- `currently / right now / at the moment` → 现在进行
- `yesterday / last week / ago / in 2020` → 一般过去
- `tomorrow / next / soon` → 将来

#### ④ 连词 vs 介词
**判断**：空格后是**句子**还是**短语**？
- 句子（有主谓）→ 连词：although, because, while, if
- 短语 / 名词 → 介词：despite, because of, during, in case of

#### ⑤ 代词 / 关系词
- **主格**（he/she/they）vs **宾格**（him/her/them）vs **物主**（his/her/their）vs **反身**（himself/herself）
- 反身代词：动作的接受者 = 发出者 *(Mr. Lee prepared the report **himself**.)*
- 关系词选 who / which / whose / where → 看从句里**缺什么**

## Part 6 长文填空（16 题，4 篇 × 4 题）

### 题型构成
每篇 4 题中通常包含：
- 1–2 道**词汇题**（看上下文选词）
- 1 道**语法题**（时态、连词、代词）
- 1 道**句子插入题**（最难）

### 句子插入题（New Type）—— 错误率最高

**解法**：
1. **看空格前后句子**，分析**话题衔接**
2. 看选项里的**代词、定冠词、连接词**：
   - *this / that / these / those + 名词* → 必须指代前文出现过的东西
   - *however / therefore / also* → 必须有逻辑对应关系
   - *the + 名词* → 这个名词前面必须出现过
3. **时间线**：选项中的时间 / 状态要符合上下文

**易错**：
- 选了"内容相关但逻辑断裂"的句子
- 选了与下一句重复的句子
- 忽视代词指代不一致

### 时态衔接
Part 6 时态题必须看**整段时态**，不能只看一句。整篇用一般现在时讲规章 → 这题也大概率现在时。

## Part 7 阅读理解（54 题）—— 决定分数上限

### 题型结构
- **单篇文章**（Single Passage）：10 篇 × 2–4 题 = 29 题
- **双篇文章**（Double Passage）：2 组 × 5 题 = 10 题
- **三篇文章**（Triple Passage）：3 组 × 5 题 = 15 题

### 文体清单
邮件、信件、公告、备忘录、广告、通知、文章、网页、表格、图表、收据、行程、对话/短信（chat / text message）。

### 题型与对应技巧

#### ① 主旨题（What is the purpose of the article? / Why was the email written?）
- 答案 90% 在**第一段**或**主题行（subject line）**
- 信号词：*I'm writing to / Please be advised that / We are pleased to announce*

#### ② 细节题（According to the article, what …?）
- **定位词**：人名、地名、日期、数字、专有名词、大写词
- 用定位词扫文章 → 找到那一句 → **同义改写**对应选项

#### ③ 推断题（What is suggested / implied / most likely true about …?）
- 答案**不会**直接写在文中
- 但必须有**文本依据** —— 一定能在原文找到对应句子
- ❌ 凭常识推 / 凭外部知识推 = 错

#### ④ 词义题（The word "X" in line N is closest in meaning to …）
- 把原句的 X 换成选项试读
- **不要**只看词典常见义；TOEIC 考的是**语境义**

#### ⑤ 句子插入题（In which of the positions [1], [2], [3], [4] does the following sentence best belong?）
- 看插入句里的 **代词 / 定冠词 / 连接词**
- 找上下文里被指代的对象

#### ⑥ 意图题（At 10:15 AM, what does X mean when she writes "..."?）
- **聊天对话题**专属
- 看**前一句**是什么问题或情境，再看 "…" 是什么反应

#### ⑦ NOT 题（What is NOT mentioned …? / Which is NOT true …?）
- 逐项排除：能在原文找到 → 排除；剩下的就是答案
- **最费时**，建议放最后做

#### ⑧ 多篇交叉题（双篇 / 三篇专属）
- 题目里**两个限制条件**来自不同篇章
- 例：第一篇说"Gold 会员享 20% 折扣"，第二篇说"Mr. Lee 是 Gold 会员且购买了 $200" → 答案是 $160
- **必做**：列出"题问什么 → 哪一篇有 → 哪一篇补充"

# 第三部分：阅读技巧专项总结

## 一、时间分配（75 分钟，100 题）

| 部分 | 题数 | 建议时间 | 累计 |
|---|---:|---:|---:|
| Part 5 | 30 | 10 min | 10 |
| Part 6 | 16 | 8 min | 18 |
| Part 7 单篇 | 29 | 25 min | 43 |
| Part 7 双篇 | 10 | 10 min | 53 |
| Part 7 三篇 | 15 | 20 min | 73 |
| 检查 | — | 2 min | 75 |

**真理**：阅读做不完是常态。**Part 7 是分数大头**，所以 Part 5 必须节省时间。

## 二、Part 7 速读策略

### 1. **题目优先 vs 文章优先？**
- **推荐：先题目，再文章**
- 步骤：
  1. 扫题目（不看选项），划出**定位词**
  2. 读文章，看到定位词就**停下来对题**
  3. 主旨题 / NOT 题 / 推断题最后做

### 2. **跳读 vs 全读？**
- **邮件、信件、通知** → **必须全读**（短）
- **文章、网页** → **跳读**：标题 + 每段首句 + 关键词
- **图表 / 表格** → **看标题 + 表头**，按题定位

### 3. **同义改写识别（最核心技能）**

TOEIC 阅读 = 同义改写测试。下表是高频对应：

| 原文 | 选项 |
|---|---|
| purchase / acquire | buy |
| inquire / query | ask |
| respond / reply | answer |
| commence / initiate | begin / start |
| terminate / conclude / cease | end / stop |
| assist / aid | help |
| modify / revise / amend / alter | change |
| relocate | move |
| proceed / continue | go on |
| complimentary / free of charge | free |
| malfunction / out of order | not working / broken |
| in advance / beforehand | early / before |
| be required to | must / need to |
| be eligible for | qualify for |
| postpone / put off / reschedule | delay |
| consult | discuss with / talk to |
| reduce / cut back | lower / decrease |
| approximately / roughly | about |
| numerous / several / various | many |
| extensive | wide-ranging |
| concerns / issues | problems |
| upcoming | next / future |
| expedite | speed up |
| renowned / prominent | famous / well-known |

### 4. **段落标记法（双/三篇必用）**

读双篇/三篇时，**在每段旁边写 2-3 个字总结**，例如：
- 第一篇：[公司发邮件] [说8月有培训] [需注册]
- 第二篇：[员工回复] [选周三场] [有冲突]
- 题目"员工最可能哪天参加" → 直接定位第二篇

### 5. **图表 / 表格题**
- 重点看**例外**：notes、asterisk (*)、conditions、except、unless
- 数字题 → 关注**单位**：thousand vs million，per unit vs total

## 三、Part 5 提速技巧

### 1. **看到选项秒判题型**

| 选项形态 | 题型 | 解法 |
|---|---|---|
| 同根词四件套 | 词性题 | 看空格语法位置 |
| 同时态动词（is/was/has been/will be） | 时态题 | 找时间状语 |
| 介词 + 连词混合 | 连介词题 | 看空格后是句子还是短语 |
| 含义不同的同词性词 | 词汇题 | 看搭配 |
| 代词 he/his/him/himself | 代词题 | 看在句中作用 |

### 2. **常见信号词速查**

**时态信号**：
- 现在完成：`since, for, so far, recently, lately, over the past, already, yet, ever`
- 现在进行：`currently, at the moment, right now`
- 将来完成：`by + 将来时间`
- 一般过去：`yesterday, ago, last, in 2020`

**主谓一致信号**：
- `one of + 复数 → 单数动词`
- `the number of → 单数 / a number of → 复数`
- `each / every / either / neither → 单数`

**虚拟语气信号（that 从句用动词原形）**：
- `suggest, recommend, propose, demand, request, insist, require`
- `It's essential / important / necessary / vital / imperative that …`

### 3. **遇到不会的：30 秒原则**
猜一个 → 标记 → 走人。**不要纠缠**，阅读时间是分数命脉。

## 四、Part 6 句子插入题加速法

### 4 步法
1. **读完全篇**（80 词左右，30 秒）
2. **看空格前一句**：在说什么？
3. **看空格后一句**：在说什么？
4. **选项**：哪个能**承上启下**？

### 信号词法
- 选项里有 `However / But` → 前文必须是相反观点
- 选项里有 `Therefore / As a result` → 前文必须是原因
- 选项里有 `Also / Moreover / In addition` → 前文必须是同方向
- 选项里有 `For example` → 前文必须是一般陈述
- 选项里有 `This / These / Such` → 必须有指代对象
- 选项里有 `the + 具体名词` → 这个名词前文必须出现过

## 五、双 / 三篇文章答题技巧

### 标准流程
1. **快速扫第一篇的开头和标题** → 确定文章类型
2. **读题目**（不看选项），判断每题问什么
3. **逐题定位**：
   - 单篇题：在某一篇找
   - 交叉题：先看哪篇有"硬信息"（数字、日期、规则），再到另一篇匹配条件

### 交叉题信号
- "Based on the email and the schedule, ..."（题干就告诉你跨篇）
- 题目问"为什么 X 收到了/没收到 Y"（需要在 A 篇看 X 状况，在 B 篇看 Y 规则）
- 题目问"X 实际付了多少钱"（A 篇规则 + B 篇消费 = 答案）

### 三篇文章常见组合
- **邮件 + 邮件回复 + 第三方资料（订单/规则/广告）**
- **广告 + 询问 + 回应**
- **公司通知 + 员工反馈 + 修订通知**

## 六、考试当天的临场策略

1. **听力部分** —— 提前**读题目**：Part 3/4 在 Directions 时间和题目间空隙抓紧扫题
2. **Part 5 节奏** —— 决不超过 10 分钟，超时立即跳
3. **Part 7 不要回头** —— 一篇做完确认涂卡，**绝不再返工**（除非提前完成）
4. **涂卡** —— 建议每做完一篇就涂；不要全部留到最后（来不及！）
5. **遇到完全不懂的文章** —— 直接看题目，按定位词找句子，文章逻辑不重要
