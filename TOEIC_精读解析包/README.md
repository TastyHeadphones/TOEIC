# TOEIC 精读解析包

这个目录现在改为“原题全文可读版”。三个主 Markdown 文件都按 PDF 页码逐页抽取了原文文本，包含题干、选项、阅读文章、听力试题册中印出的题目，以及答案册/解析区能抽取到的中文解析。

使用方式：

- 直接在 Markdown 内搜索题号，例如 `123.`、`Questions 186-190`、`Test 03`。
- 每个页面都有 `PDF 第 N 页` 标题，方便回到原 PDF 对照版面。
- 图片、照片、图表无法可靠转成纯 Markdown；这些页会保留页码提示，需要看原 PDF。
- `托业新题型试题集.pdf` 与 `托业新题型试题集（可复制版本）.pdf` 内容对应；全文抽取使用可复制版，普通版作为版面参照。

## 文件

- `00_高频问题专项整理.md`：所有题目反推的高频问题。
- `01_听力1000题逐题解析索引.md`：听力1000题 PDF 全文抽取版，含试题册与答案册文本。
- `02_阅读1000题逐题解析索引.md`：阅读1000题 PDF 全文抽取版，含试题册与答案册文本。
- `03_新题型600题逐题解析索引.md`：新题型600题 PDF 全文抽取版。

## 规模

- 听力1000题：1000题
- 阅读1000题：1000题
- 新题型600题：600题
