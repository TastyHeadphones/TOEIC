# TOEIC 精读解析包

这个目录是按书和题号生成的精读索引。为避免直接复制整本题库，逐题条目保留答案、题型、考点、解题动作和极短定位信息；完整题干和选项请回到原PDF对应题号查看。

`托业新题型试题集.pdf` 与 `托业新题型试题集（可复制版本）.pdf` 内容对应；普通版文本不可稳定抽取，因此逐题索引使用可复制版，普通版作为版面参照。

## 文件

- `00_高频问题专项整理.md`：所有题目反推的高频问题。
- `01_听力1000题逐题解析索引.md`：听力1000题全部1000题索引。
- `02_阅读1000题逐题解析索引.md`：阅读1000题全部1000题索引。
- `03_新题型600题逐题解析索引.md`：新题型600题索引；少数OCR缺失标为待核对。

## 规模

- 听力1000题：1000题
- 阅读1000题：1000题
- 新题型600题：600题
